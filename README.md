# cogssg
A simple static site generator

Powered by [cog](https://github.com/nedbat/cog).

## Installing

```shell
pip install cogssg
```

## Building

```shell
make <target>
```

### Available targets

| Target   | Description                      |
|----------|----------------------------------|
| `all`    | Builds the package.              |
| `upload` | Uploads the package to pypi.org. |
